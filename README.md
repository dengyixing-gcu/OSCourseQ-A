# 操作系统课程知识问答智能体

《操作系统》课程本科学生自学辅助工具，覆盖设备管理（第7章）和文件管理（第8章）两大核心模块。

## 功能特性

- **智能问答**：支持自然语言提问，自动理解学生意图并返回准确答案
- **知识导航**：侧边栏按章节组织知识点，方便浏览和查阅
- **双章覆盖**：设备管理（I/O管理、磁盘管理、缓冲管理）和文件管理（文件系统、目录、保护机制）
- **纯前端实现**：无需后端服务，浏览器直接打开即可使用
- **多级匹配**：精确关键词匹配 → 同义词扩展匹配 → TF-IDF语义相似度匹配

## 使用方法

1. 直接打开 `index.html` 文件
2. 在输入框中用自然语言提问（如"什么是DMA控制方式？"）
3. 点击发送或按回车键获取答案
4. 点击顶部标签切换章节（设备管理/文件管理）
5. 点击侧边栏知识点直接查看内容
